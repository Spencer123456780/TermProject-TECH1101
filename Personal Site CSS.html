CSS Cheet Sheet

This is web page made for the ease of finding CSS properties.


What is CSS? CSS stands for Cascade Styling Sheet. CSS is the styling of a webpage.


    Attributes:
  • *. The star attribute effects the entire webpage.
  • Body. The body attribute effects the body of the page.
  • #. The hashtage tagrest any id set in the html file then name comes after the hashtag.
  • "." The period effects any div that was set in the html file.

    Background:
  • Background-Image. Allows an image to be set as the bakground
  • Background-Position. Allows you to change the postion of the background image
  • Background-Size. Allows the size of the background to be changed
  • Background-Repeat. Allows the background to change its reapting options for static background set to no-repeat
  • Background-Color. Allows you to set a color as your background

    Size:
  • Hight. Hight allows you to set the Hight of any object. It can be set to px or to %
  • Width. Width allows the width of any object to be set similar to hight
  • Max-Width. Sets the max width of an Object
  • Min-Hight. Sets the max hight of an Object

    Margin:
  • Margin-Left. Allows you to add a marge to the left side of the targeted elemment. remcomend setting in % for dynamic size change
  • Margin-Right. Allows you to set the margin of the right side the elemment
  • Margin-Top. Allows the top of the element to be set
  • Margin-Bottom. Allows the bottom of the elemment to be set

    Padding:
  • Padding-Top. Adds padding to the top of an elemment. Padding is like extra blank space. Remcomend using % over px
  • Padding-Bottom. Allows padding at the bottom of an elemment to be set
  • Padding-Right. Adds padding to the right side of the targeted elemment
  • Padding-Left. Adds padding to the left of the elemment

    Border:
  • Border-Top. Allows a border to be set at the top of an elemment. Border is a line that is created around an elemment not replacing anything but adding.
  • Border-Bottom. Allows a border to be set at the bottom of an elemment
  • Border-Right. Adds a border to the right side of the elemment
  • Border-Left. Adds a border to the left side of the elemment
